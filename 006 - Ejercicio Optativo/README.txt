# Proyecto Integrador: Análisis de Imagen

Este proyecto tiene como objetivo aplicar herramientas y conceptos aprendidos en la materia Procesamiento de Imágenes Digitales

## Archivos del Proyecto

- `integrador_toys.ipynb`: Notebook principal del proyecto. Contiene todo el flujo de trabajo de análisis de datos:
  - Importación de librerías
  - Carga y exploración inicial de los datos
  - Limpieza y transformación
  - Análisis exploratorio y visualización de los resultados
  - Conclusiones


## Cómo usar el código

1. Clonar este repositorio o descargar el archivo `.ipynb`.
2. Abrir el notebook con Jupyter Notebook o Google Colab.
3. Ejecutar las celdas paso a paso.
4. Si el notebook requiere archivos externos (como un CSV), asegurarse de tenerlos disponibles en la misma carpeta o subirlos al entorno si se usa Colab.
