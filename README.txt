# Nivelación en Python

Este notebook está diseñado como un repaso y nivelación de los conceptos fundamentales de programación en Python. Es ideal para quienes comienzan en el mundo del análisis de datos o desean refrescar conocimientos básicos antes de adentrarse en contenidos más avanzados.

## Este proyecto pertenece a Sebastián Emanuel Flehr estudiante del IFTS24 realizado el 19 de Abril del 2025

## Contenidos

- Variables y tipos de datos
- Operaciones básicas
- Estructuras de control:
  - Condicionales (`if`, `else`, `elif`)
  - Bucles (`for`, `while`)
- Funciones
- Listas, tuplas y diccionarios
- Manejo de errores (`try`, `except`)
- Introducción a módulos y librerías (`math`, `random`, etc.)

## Requisitos

- Python 3.x
- No se requiere instalación adicional si se utiliza Google Colab

## Cómo usar

1. Cloná este repositorio o abrí el notebook directamente en Google Colab.
2. Ejecutá las celdas secuencialmente.
3. Modificá o experimentá con el código para afianzar conocimientos.

## Público objetivo

- Estudiantes de carreras técnicas o científicas
- Participantes de cursos de ciencia de datos
- Personas que se están iniciando en la programación con Python
